# Learning Python
A collection of Python projects and exercises built while learning Python from scratch.

## Projects
### Weather App
- Built a CLI weather app that fetches real-time weather data using an API
- Tech used: Python, Requests library, OpenWeatherMap API

### IPL Stats Analyzer
- Analyzed IPL data (2008-2024) using Pandas and NumPy across 10 analytical questions
- Covered team win rates, phase-wise strike rates, bowler consistency (std dev of economy),
  toss correlation, custom vectorized impact score, Orange Cap winners, Super Over stats
- Tech: Python, Pandas, NumPy, Matplotlib
